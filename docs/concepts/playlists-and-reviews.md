# Playlists And Reviews

Playlists organize clips into review material.

A single clip captures a moment. A playlist tells a story across moments.

## Common Playlist Types

- player review
- unit review
- opponent pattern
- phase of play
- tactical concept
- training feedback

## Good Review Structure

Good playlists usually have:

- a clear title
- clips in a deliberate order
- enough context for someone else to understand the theme
- tactical annotations when filtering or comparison matters

## Practical Rule

Build playlists when individual clips start pointing to the same coaching question.
