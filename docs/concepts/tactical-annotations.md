# Tactical Annotations

Tactical annotations add football meaning to clips, events, or time ranges.

They are different from free-form tags. They should make analysis queryable, reviewable, and shareable.

## What They Capture

Start with the smallest useful structure:

- phase
- concept
- outcome
- optional note

Examples:

- `in possession / line break / success`
- `out of possession / pressing trigger / fail`
- `defensive transition / recovery run / neutral`

## What They Attach To

Attach tactical annotations to:

- clips, when the moment is review material
- events, when the action is the anchor
- time ranges, when the tactical idea spans multiple actions

## Practical Rule

Use tactical annotations when the question is "what does this moment mean?" rather than "what happened?"
