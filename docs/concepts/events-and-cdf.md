# Events And CDF

Events describe football actions or moments. They can come from provider imports, manual annotation, or future automated workflows.

CDF, the Common Data Format for Football, is a structured format for exchanging football event data.

## Events In Threader

Events can help you:

- navigate the timeline
- create clips from known actions
- filter match moments
- validate analysis data
- prepare exports

## CDF In Threader

CDF is important for interoperability, but it is not the only reason to use Threader.

You can use Threader to build clips, tactical annotations, playlists, and feed discussions even when you do not plan to export CDF data.

## Practical Rule

Import event data when you have it. If you do not, start from the video and create clips from time ranges.
