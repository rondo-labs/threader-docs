# Projects

A project is Threader's local analysis container.

Use one project for one match, training session, or focused review context. The project keeps the football analysis objects connected so the video, events, clips, annotations, drawings, and playlists can refer to the same moments.

## A Project Can Contain

- match video references
- imported or manual events
- clips
- drawings
- tactical annotations
- playlists and review structure
- local project state

## Project Data Is Local

Project data is the foundation of Threader's desktop workflow.

Cloud collaboration data, such as workspace membership and feed comments, is separate until you explicitly share or publish analysis material.

## Practical Rule

Start with a project and a video. Add events, clips, annotations, and playlists as the analysis becomes more structured.
