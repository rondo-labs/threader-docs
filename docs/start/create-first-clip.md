# Create Your First Clip

Use this when you have found a moment that should become review material.

A clip is a saved video range with enough context to revisit, annotate, group, and share.

## Create From A Time Range

1. Find the moment in the video.
2. Select the start and end range on the timeline.
3. Create a clip from the selection.
4. Adjust the range if needed.
5. Give the clip a clear title.

This path works even when the project has no imported event data.

## Create From An Event

1. Select an event in the timeline or event list.
2. Create a clip from the event.
3. Adjust pre-roll and post-roll if needed.
4. Add a title that explains why the moment matters.

## After Creating A Clip

Next actions:

- add tactical annotation
- add drawing or telestration
- group the clip into a playlist
- publish the clip or playlist to feed when ready

Good clip titles describe the football idea, not only the event type.
