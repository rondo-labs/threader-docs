# Import Match Video

Use this when a project has no video yet.

Video gives Threader the analysis surface. Events, clips, drawings, and annotations become easier to understand when they can stay connected to the match footage.

## Import A Video

1. Open or create a project.
2. Choose the video import action.
3. Select the local match video file.
4. Wait for Threader to read metadata.
5. Confirm that playback works.

Threader is designed for common match video formats such as MP4, MOV, and MKV.

## If Import Fails

Transcode the file to a standard H.264 MP4 and try again.

Very large files, unusual codecs, or files stored on unavailable external drives can cause playback or metadata issues.

## After Import

Next actions:

- navigate the video with the player and timeline
- import event data if available
- select a time range and create a clip
- set match time sync when you need CDF-ready export

You do not need event data before creating your first clip.
