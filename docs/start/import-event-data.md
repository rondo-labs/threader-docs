# Import Event Data

Use this when you already have match events from a provider, data file, or previous workflow.

Imported events help Threader connect moments on the timeline to the video. They can become clip starting points, filters, and review context.

## Import Events

1. Open the project that contains the match video.
2. Choose the event import action.
3. Select the supported provider or file format.
4. Review any mapping or validation feedback.
5. Confirm the import.

## After Import

Use imported events to:

- jump to moments on the timeline
- create clips from selected events
- filter dense passages
- connect tactical annotations to specific match actions
- prepare CDF export when needed

## If You Do Not Have Events

You can still analyze the video.

Create clips from time ranges first, then add tactical annotations or manual event context later.
