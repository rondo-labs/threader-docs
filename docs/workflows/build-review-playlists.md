# Build Review Playlists

Use this when several clips belong in the same coaching or analysis story.

A playlist turns individual clips into review material. It can be organized by player, phase, tactical concept, opponent behavior, or meeting topic.

## Steps

1. Select clips that belong together.
2. Create a playlist.
3. Give the playlist a clear review title.
4. Reorder clips into the sequence you want to present.
5. Add notes or context where needed.

## Good Playlist Titles

- `High press recoveries`
- `Final-third cutbacks`
- `Player 8 receiving under pressure`
- `Defensive transition problems`

## Before Sharing

Check that each clip has:

- a clear range
- a readable title
- tactical meaning when needed
- enough context for someone else to understand it
