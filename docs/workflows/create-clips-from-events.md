# Create Clips From Events

Use this when imported or manual events already identify the moment you want to review.

Event-based clips keep the clip connected to structured match data.

## Steps

1. Select an event from the timeline or event list.
2. Create a clip from that event.
3. Adjust the clip range around the event.
4. Confirm the title and linked event.
5. Add tactical annotation if the moment needs football meaning.

## When This Works Best

- provider data already marks the action
- the event is the anchor for review
- you need to filter or group clips by event type later
- you plan to export or validate structured data

## Range Tips

Do not make every event clip too tight. Coaching review often needs the build-up before the action and the consequence after it.
