# Import Video

Start each project by importing the match video you want to analyze.

## Steps

1. Open or create a Threader project.
2. Choose the video import action from the app.
3. Select a local video file.
4. Wait for Threader to read the file metadata.
5. Confirm that playback works before adding annotations.

## Supported Files

Threader is designed for common match video formats such as MP4, MOV, and MKV. If a file does not import cleanly, transcode it to a standard H.264 MP4 and try again.

## After Import

Once the video is available, set up match metadata and whistle sync before final export. You can still annotate first, but sync should be completed before generating CDF output.
