# Create Clips From Time Ranges

Use this when you know the video moment but do not have a specific event to start from.

Time-range clips are the fastest path from watching video to saving review material.

## Steps

1. Find the moment in the video.
2. Select the start and end range on the timeline.
3. Create a clip from the selection.
4. Adjust the start and end time.
5. Add a clear title.

## When This Works Best

- you do not have imported event data
- the moment spans several events
- you are reviewing a tactical pattern rather than a single action
- you want to capture pre-roll and post-roll manually

## Next Actions

- add tactical annotation
- add a drawing
- add the clip to a playlist
- share the clip when the review context is ready
