# Export CDF

CDF export turns a Threader project into structured football data that can be used outside the app.

## Before You Export

Check that the project has:

- A video imported and linked
- Match metadata
- Teams and player records when player-level events are used
- Whistle sync points for reliable timestamps
- Events with required CDF fields

## Export Flow

1. Open the export panel.
2. Run validation.
3. Review any warnings or errors.
4. Choose the export destination.
5. Export the CDF files.
6. Open the destination folder and inspect the output.

## Validation

Validation is part of the export workflow. Treat validation errors as blockers and warnings as review prompts.
