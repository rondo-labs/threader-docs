# Import Match Data

Threader can import provider event data and display it alongside your own analysis. Current adapter code includes support paths for StatsBomb, Opta, Second Spectrum, and Wyscout files.

## Import Flow

1. Open the import workflow.
2. Choose a provider or let Threader detect the file when supported.
3. Select the data file.
4. Preview the import.
5. Attach the import to the current video.
6. Toggle visibility or color once imported.

## Why Import Provider Data

Imported data can help you compare manual annotations with an external feed, jump to important moments faster, or enrich player and event review workflows.

## Cleanup

If an import is no longer useful, remove the import session from the project. This leaves your manually created Threader events intact.
