# Export CDF Data

Use this when you need structured football event data outside Threader.

CDF export is an output path, not the only goal of Threader. Many users will build clips and review playlists without exporting data.

## Before Export

Check that the project has:

- match video
- match time mapping if required
- valid event records
- required teams and players
- resolved validation warnings

## Export

1. Open the export workflow.
2. Review validation feedback.
3. Fix missing required fields.
4. Export the CDF-ready data.
5. Use the exported files in downstream analysis tools or pipelines.
