# Annotate Events

Threader is optimized for keyboard-first event annotation. The goal is to keep your attention on the match while capturing structured CDF data.

## Basic Flow

1. Play the video.
2. Trigger the event type with a shortcut or command.
3. Confirm the default fields.
4. Add player, team, coordinates, or qualifiers when needed.
5. Press Enter to save the event.
6. Review the event in the timeline.

## Tips for Speed

- Use slower playback for dense passages.
- Tag the primary event first, then add secondary details during review.
- Keep roster data ready before annotating player-heavy workflows.
- Use the timeline filters to audit one event type at a time.

## Quality Pass

Before export, review events for missing teams, players, coordinates, and sync data. The validation panel and CDF export checks are designed to catch these issues.
