# Timeline Basics

Use this when you want to understand where video, events, clips, and annotations meet.

The timeline is Threader's central analysis surface. It is not only a playback bar. It helps you move between match footage, imported events, saved clips, drawings, and tactical meaning.

## What The Timeline Connects

- video playback position
- match events
- selected time ranges
- clips
- tactical annotations
- review moments

## Common Actions

- seek to a moment in the video
- inspect events around the current playhead
- select a time range
- create a clip from the selection
- create a clip from an event
- review dense passages before building playlists

## Recommended First Workflow

1. Import match video.
2. Move to an important moment.
3. Select the useful time range.
4. Create a clip.
5. Add tactical meaning if the clip will be reviewed or shared.

Imported events make the timeline richer, but they are not required for first clip creation.
