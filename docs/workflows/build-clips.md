# Build Clips

Use clips to turn annotated match moments into review material.

## Create a Clip

1. Find the moment in the timeline or video player.
2. Create a clip around the event or time range.
3. Adjust the start and end time.
4. Add player tags or linked events.
5. Add drawings when the clip needs tactical context.

## Build a Playlist

1. Select the clips that belong together.
2. Create a playlist.
3. Reorder clips into the review sequence.
4. Export the playlist when it is ready to share.

## Naming

Use names that survive outside the app, such as `High press recoveries - first half` or `Player 8 receiving under pressure`.
