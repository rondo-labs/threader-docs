# Add Tactical Annotations

Use this when a clip or moment needs football meaning beyond an event label.

Tactical annotations connect clips, events, or time ranges to structured football concepts. They make moments easier to filter, review, and discuss.

## Minimum Useful Annotation

Start small:

- phase
- concept
- outcome
- short note if needed

You do not need to fill every possible detail before the annotation is useful.

## What To Attach

Attach the annotation to the most useful analysis object:

- a clip, when the moment is review material
- an event, when a specific action matters
- a time range, when the tactical idea spans several actions

## Good Annotation Notes

Good notes explain why the moment matters:

- `pressing trigger after backward pass`
- `fullback receives isolated with no support`
- `counter-press fails after loose second ball`

## After Annotation

Use annotations to:

- filter clips by tactical concept
- build playlists around repeated patterns
- publish review material with more context
- compare successful and failed outcomes
