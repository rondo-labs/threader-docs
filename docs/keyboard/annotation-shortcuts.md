# Annotation Shortcuts

Use annotation shortcuts when you are adding structured football meaning repeatedly.

## Common Annotation Jobs

- open tactical annotation editor
- confirm the current annotation
- move between fields
- reuse recent values when supported
- return to the clip or timeline context

## Tip

Do not over-annotate every moment. Add structured tactical meaning when it will help filtering, review, or sharing.
