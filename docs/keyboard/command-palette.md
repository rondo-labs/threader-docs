# Command Palette

Use this when you know what you want to do but do not remember where the control lives.

The command palette is the fastest way to find actions, open Help topics, and move through Threader without hunting through panels.

## Common Uses

- import video
- create a clip from the current selection
- open timeline actions
- open keyboard shortcuts
- open Getting Started
- find workspace or feed actions

## Practical Rule

If you are lost, open the command palette and search for the job you want to do.
