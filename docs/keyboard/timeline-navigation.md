# Timeline Navigation

Use timeline navigation when you need precise movement through events, clips, and time ranges.

## Common Navigation Jobs

- move to the current video position
- jump between events
- inspect nearby clips
- adjust selected ranges
- return to an important moment

## Tip

Use the command palette when you cannot remember a timeline shortcut. Search for the action by name.
