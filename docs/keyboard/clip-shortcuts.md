# Clip Shortcuts

Use clip shortcuts when you are reviewing video and saving moments quickly.

## Common Clip Jobs

- create clip from selected time range
- create clip from selected event
- adjust clip start and end
- add clip to playlist
- open clip details

## Tip

Create clips first, then add tactical meaning only to the moments that will be reviewed, compared, or shared.
