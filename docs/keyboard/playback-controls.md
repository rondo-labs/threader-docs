# Playback Controls

Use playback controls to move through match video without leaving the analysis flow.

Threader is designed for keyboard-first review, especially when tagging dense passages or building clips.

## Use Playback Shortcuts For

- play and pause
- seek forward or backward
- review around the current moment
- slow down dense sequences
- return to normal speed

## Tip

When creating clips or annotations, slow playback before the moment and then refine the range on the timeline.
