# Roster Management

Roster management stores the teams and players that make annotation, clip review, and CDF export more complete.

## Tools

- Player panel
- Lineup editor
- Roster import dialog
- Team and player library flows
- Player detail views for linked events and clips

## Practical Workflow

Set up rosters before a full annotation pass. That keeps player tagging fast and reduces cleanup during export validation.
