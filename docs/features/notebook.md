# Notebook

Threader includes notebook-oriented tools for analysis inside the project workspace. The notebook feature is intended for analysts who want to inspect project data, run analysis steps, and turn results back into review assets.

## What You Can Use It For

- Explore project data
- Run analysis cells
- Inspect variables and outputs
- Create playlists from selected events
- Keep notes close to the match context

## Setup

Some notebook workflows may require a local Python setup. Use the in-app setup guide when Threader prompts for missing runtime pieces.
