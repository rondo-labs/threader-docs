# Command Palette

The command palette gives Threader a keyboard-driven control layer similar to modern desktop tools. Use it when you know what you want to do but do not want to move through panels.

## Common Uses

- Start an annotation action
- Switch workflows
- Find a command without memorizing every shortcut
- Keep your hands on the keyboard during video review

## Shortcut Strategy

Use dedicated shortcuts for actions you perform constantly. Use the command palette for less frequent actions that still need to be quick.
