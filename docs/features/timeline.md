# Timeline

The timeline is the review surface for events, imported data, drawings, clips, and player-focused analysis.

## What It Helps With

- Navigate to annotated moments
- Filter dense event sequences
- Compare imported provider events with manual tags
- Review tactical annotations over time
- Jump from a player to related events and clips

## Review Pattern

Use the timeline after an annotation pass to audit consistency. Filter down to one event type or player, then move through each item and correct missing fields.
