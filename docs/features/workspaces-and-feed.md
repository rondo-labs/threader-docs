# Workspaces and Feed

Workspaces and feed tools support collaboration around analysis material.

## Workspaces

Workspace settings include profile, members, and invitations. Use them when multiple people need access to shared analysis workflows.

## Feed

The feed supports rich-text posts, comments, mentions, and sharing selected analysis material. Use it to discuss moments without separating the conversation from the match context.

## Account

Account tools handle sign-in, invitations, and profile state for collaborative workflows.
