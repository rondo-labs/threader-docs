# Missing Media

Use this when a project opens but the video is unavailable.

Projects can reference local media files. If the file moved, was renamed, or lives on an unavailable drive, Threader may need the media to be reconnected.

## Try First

1. Confirm the video file still exists.
2. Connect the external drive if the file lives there.
3. Restore the original file path if possible.
4. Reconnect the media from inside Threader when prompted.

## What Is Still Safe

Missing media does not mean the project data is gone. Events, clips, annotations, and playlists can still exist, but video playback needs the media file.
