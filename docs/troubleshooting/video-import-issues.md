# Video Import Issues

Use this when a video will not import, load, or play correctly.

## Try First

1. Confirm the file exists on a local or available drive.
2. Try a standard MP4 version of the file.
3. Avoid unusual codecs when possible.
4. Reopen the project and reconnect missing media if prompted.

## Recommended Format

If a file is unreliable, transcode it to H.264 MP4 and import that version.
