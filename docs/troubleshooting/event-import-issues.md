# Event Import Issues

Use this when event data does not import cleanly or does not align with the match.

## Check

- the provider or file format is supported
- required fields are present
- team and player identifiers can be mapped
- event time uses the expected clock or timestamp basis
- the imported match matches the project video

## If Events Are Not Ready

You can still create clips from video time ranges and add event context later.
