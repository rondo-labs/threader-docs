# Workspace And Feed Issues

Use this when collaboration state does not look right.

## Check

- you are signed in
- you are in the expected workspace
- the invitation was accepted with the correct account
- the item was actually published to feed
- the local project has the content you are trying to share

## Remember

Local project data and cloud collaboration data are separate. Creating a clip locally does not publish it automatically.
