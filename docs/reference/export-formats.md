# Export Formats

Threader has two main export surfaces: CDF data export and video clip export.

## CDF Data Export

Use CDF export when you need structured event data for analysis pipelines, validation, or interoperability with other football data tools.

Expected export targets include CDF-oriented event records and related project metadata.

## Clip Export

Use clip export when you need video deliverables for review, coaching, or presentation. Clip exports can be created from individual clips or playlists.

## Validation

Run validation before CDF export. For clip export, review the clip boundaries and linked drawings before rendering.
