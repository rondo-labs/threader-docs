# Supported Event Types

Use this as the reference page for event types that Threader can import, annotate, validate, or export.

This page should stay aligned with the app's current event model and supported provider adapters.

## Common Football Event Families

- pass
- shot
- carry or dribble
- duel
- tackle
- interception
- clearance
- goalkeeper action
- set piece
- foul
- card
- substitution

## Note

Provider event names may differ from Threader's normalized internal event model. Import adapters should normalize provider-specific shapes before events reach the main analysis UI.
