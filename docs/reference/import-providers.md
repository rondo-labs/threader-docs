# Import Providers

Threader includes import adapter code for common football data providers.

## Provider Adapters

| Provider | Typical use |
| --- | --- |
| StatsBomb | Event data import |
| Opta | Event data import |
| Second Spectrum | Tracking or event-derived workflows |
| Wyscout | Event data import |

## Import Sessions

Imported files are stored as import sessions. A session can be shown, hidden, recolored, inspected, or deleted without removing your manually created Threader annotations.

## Best Practice

Import provider data after the video is linked, so Threader can attach the imported session to the correct video record.
