# Keyboard Shortcuts

Threader is designed for keyboard-first review. This reference will grow as shortcuts are finalized.

## Global

| Shortcut | Action |
| --- | --- |
| `~` | Toggle Cinema Mode |

## Annotation

Use event shortcuts for actions you tag repeatedly. If you do not remember a shortcut, open the command palette and search for the action.

## Video Review

Use video controls for playback, frame review, and speed changes during dense passages. Keep playback slower when tagging locations or tactical details.
