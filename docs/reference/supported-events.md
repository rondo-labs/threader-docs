# Supported Event Types

Reference of all event types and sub-types supported by Threader.

Threader stores annotations as CDF-oriented football event records. The exact supported event set will evolve with the product, but the reference model should be read in three layers:

- **Event type**: the primary football action, such as pass, shot, duel, recovery, substitution, or set piece.
- **Subtype and qualifiers**: details that make the event useful for review and analysis.
- **Context**: team, player, time, coordinates, related clips, imported provider links, and validation state.

## Documentation Status

This page is the public reference placeholder for the supported event catalog. As the in-app event schema stabilizes, add the final event table here with required fields, optional fields, and CDF mapping notes.
