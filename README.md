# Threader Docs

Documentation site for **Threader** — a desktop football video annotation tool for coaches and analysts.

Built with [Docusaurus](https://docusaurus.io/) and deployed on [Vercel](https://vercel.com/).

## Development

```bash
pnpm install
pnpm start
```

## Build

```bash
pnpm build
```
