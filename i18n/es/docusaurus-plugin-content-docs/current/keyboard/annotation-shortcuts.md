# Atajos de annotation

Usa atajos de annotation cuando añadas significado futbolístico estructurado repetidamente.

## Tareas comunes de annotation

- abrir el tactical annotation editor
- confirmar la annotation actual
- moverte entre campos
- reutilizar valores recientes cuando esté soportado
- volver al contexto de clip o timeline

## Consejo

No sobre-anotes cada momento. Añade significado táctico estructurado cuando ayude a filtrar, revisar o compartir.
