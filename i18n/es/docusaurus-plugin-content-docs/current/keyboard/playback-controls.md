# Controles de reproducción

Usa los controles de reproducción para moverte por el video del partido sin salir del flujo de análisis.

Threader está diseñado para revisión keyboard-first, especialmente al etiquetar pasajes densos o construir clips.

## Usa atajos de reproducción para

- reproducir y pausar
- saltar hacia delante o atrás
- revisar alrededor del momento actual
- ralentizar secuencias densas
- volver a velocidad normal

## Consejo

Cuando crees clips o annotations, ralentiza antes del momento y luego ajusta el rango en el timeline.
