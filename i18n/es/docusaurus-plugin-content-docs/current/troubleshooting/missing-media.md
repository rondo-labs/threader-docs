# Media faltante

Usa esto cuando un proyecto se abre, pero el video no está disponible.

Los proyectos pueden referenciar archivos locales de media. Si el archivo se movió, cambió de nombre o está en un disco no disponible, Threader puede necesitar que reconectes la media.

## Prueba primero

1. Confirma que el archivo de video todavía existe.
2. Conecta el disco externo si el archivo vive allí.
3. Restaura la ruta original si es posible.
4. Reconecta la media desde Threader cuando se solicite.

## Qué sigue seguro

Media faltante no significa que los datos del proyecto hayan desaparecido. Eventos, clips, annotations y playlists pueden seguir existiendo, pero la reproducción necesita el archivo de video.
