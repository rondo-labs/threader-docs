# Problemas al importar eventos

Usa esto cuando los datos de eventos no se importan correctamente o no se alinean con el partido.

## Comprueba

- el provider o formato de archivo es compatible
- los campos obligatorios existen
- los identificadores de equipos y jugadores pueden mapearse
- el tiempo del evento usa la base de reloj o timestamp esperada
- el partido importado coincide con el video del proyecto

## Si los eventos no están listos

Todavía puedes crear clips desde rangos de tiempo del video y añadir contexto de evento después.
