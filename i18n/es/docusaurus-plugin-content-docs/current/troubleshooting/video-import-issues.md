# Problemas al importar video

Usa esto cuando un video no se importa, carga o reproduce correctamente.

## Prueba primero

1. Confirma que el archivo existe en un disco local o disponible.
2. Prueba una versión MP4 estándar.
3. Evita códecs poco comunes cuando sea posible.
4. Reabre el proyecto y reconecta media faltante si se solicita.

## Formato recomendado

Si un archivo no es fiable, transcódalo a MP4 H.264 e importa esa versión.
