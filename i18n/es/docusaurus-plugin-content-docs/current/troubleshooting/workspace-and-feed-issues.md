# Problemas de workspace y feed

Usa esto cuando el estado de colaboración no se ve correcto.

## Comprueba

- has iniciado sesión
- estás en el workspace esperado
- la invitación se aceptó con la cuenta correcta
- el item realmente se publicó en feed
- el proyecto local contiene el material que quieres compartir

## Recuerda

Los datos locales de proyecto y los datos de colaboración en la nube están separados. Crear un clip localmente no lo publica automáticamente.
