# Bienvenido a Threader

**Threader** es un espacio de trabajo de escritorio para análisis de fútbol, creado para entrenadores y analistas. Mantiene conectados el video del partido, eventos, clips, tactical annotations, playlists y material de colaboración dentro de un proyecto local.

## Qué es Threader

Threader te ayuda a pasar de la imagen del partido a material de revisión estructurado. Importa un video, conecta eventos o rangos de tiempo con clips, añade significado táctico, construye playlists y comparte los resultados con tu staff cuando estén listos.

El flujo principal es:

```text
Project -> Video -> Event / Time Range -> Clip -> Tactical Annotation -> Playlist / Review -> Feed
```

## Para quién es

- **Performance analysts** que necesitan datos estructurados de partido
- **Coaches** que quieren convertir momentos clave en material técnico
- **Video analysts** que necesitan creación rápida de clips y flujos de revisión
- **Data analysts** que necesitan eventos CDF-compliant para pipelines downstream
