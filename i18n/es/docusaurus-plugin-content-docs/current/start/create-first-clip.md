# Crea tu primer clip

Usa esto cuando encuentres un momento que debería convertirse en material de revisión.

Un clip es un rango de video guardado con suficiente contexto para volver a verlo, anotarlo, organizarlo y compartirlo.

## Crear desde un rango de tiempo

1. Encuentra el momento en el video.
2. Selecciona el rango de inicio y fin en el timeline.
3. Crea un clip desde la selección.
4. Ajusta el rango si hace falta.
5. Dale un título claro.

Este camino funciona incluso si el proyecto no tiene datos de eventos importados.

## Crear desde un evento

1. Selecciona un evento en el timeline o en la lista de eventos.
2. Crea un clip desde el evento.
3. Ajusta pre-roll y post-roll si hace falta.
4. Añade un título que explique por qué importa el momento.

## Después de crear un clip

Próximas acciones:

- añade una tactical annotation
- añade un drawing o telestration
- agrupa el clip en una playlist
- publica el clip o la playlist en feed cuando esté listo el contexto de revisión

Un buen título de clip describe la idea futbolística, no solo el tipo de evento.
