# Importa el video del partido

Usa esto cuando un proyecto todavía no tiene video.

El video le da a Threader la superficie de análisis. Eventos, clips, drawings y annotations son más fáciles de entender cuando siguen conectados con la imagen del partido.

## Importar un video

1. Abre o crea un proyecto.
2. Elige la acción de importar video.
3. Selecciona el archivo local del partido.
4. Espera a que Threader lea los metadatos.
5. Confirma que la reproducción funciona.

Threader está pensado para formatos habituales de video de partido, como MP4, MOV y MKV.

## Si la importación falla

Transcodifica el archivo a un MP4 H.264 estándar e inténtalo de nuevo.

Archivos muy grandes, códecs poco comunes o archivos guardados en discos externos no disponibles pueden causar problemas de reproducción o metadatos.

## Después de importar

Próximas acciones:

- navega el video con el reproductor y el timeline
- importa datos de eventos si los tienes
- selecciona un rango de tiempo y crea un clip
- configura la sincronización de tiempo de partido cuando necesites exportar CDF

No necesitas eventos para crear tu primer clip.
