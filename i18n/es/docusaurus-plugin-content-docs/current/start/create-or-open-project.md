# Crea o abre un proyecto

Usa esto cuando vayas a iniciar una sesión de análisis de partido.

Un proyecto de Threader es un espacio de trabajo local. Mantiene juntos el video, los eventos importados o manuales, clips, drawings, tactical annotations, playlists y el estado del proyecto.

## Crear un proyecto nuevo

1. Abre Threader.
2. Elige la acción de nuevo proyecto.
3. Nombra el proyecto según el partido, sesión de entrenamiento o tema de revisión.
4. Guárdalo en una ubicación adecuada para tus archivos de análisis.

Usa un proyecto por partido cuando el análisis dependa del video, del tiempo de partido o de datos de eventos.

## Abrir un proyecto existente

Usa proyectos recientes cuando vuelvas a un análisis activo.

Si un proyecto se abre sin su media, los datos del proyecto siguen intactos; es posible que Threader necesite que vuelvas a conectar archivos de video que se movieron.

## Qué pertenece a un proyecto

- referencias a videos del partido
- datos de eventos
- clips
- drawings
- tactical annotations
- playlists y estructura de revisión
- estado local de análisis

## Qué no pertenece solo a un proyecto

La membresía de workspace, comentarios de feed, invitaciones y datos de cuenta pertenecen a la capa de colaboración.

Los proyectos son contenedores de análisis local. Workspace y feed son superficies explícitas para compartir.
