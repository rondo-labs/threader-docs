# Importa datos de eventos

Usa esto cuando ya tienes eventos del partido desde un provider, archivo de datos u otro flujo.

Los eventos importados ayudan a conectar momentos del timeline con el video. Pueden convertirse en puntos de inicio para clips, filtros y contexto de revisión.

## Importar eventos

1. Abre el proyecto que contiene el video del partido.
2. Elige la acción de importar eventos.
3. Selecciona el provider o formato de archivo compatible.
4. Revisa cualquier feedback de mapeo o validación.
5. Confirma la importación.

## Después de importar

Usa los eventos importados para:

- saltar a momentos en el timeline
- crear clips desde eventos seleccionados
- filtrar pasajes densos
- conectar tactical annotations con acciones concretas
- preparar exportación CDF cuando haga falta

## Si no tienes eventos

Todavía puedes analizar el video.

Crea clips desde rangos de tiempo primero y añade contexto táctico o eventos manuales después.
