# Crea clips desde rangos de tiempo

Usa esto cuando conoces el momento en el video, pero no tienes un evento concreto como punto de inicio.

Los clips por rango de tiempo son la forma más rápida de pasar de ver video a guardar material de revisión.

## Pasos

1. Encuentra el momento en el video.
2. Selecciona el rango de inicio y fin en el timeline.
3. Crea un clip desde la selección.
4. Ajusta el inicio y el final.
5. Añade un título claro.

## Cuándo funciona mejor

- no tienes datos de eventos importados
- el momento abarca varios eventos
- estás revisando un patrón táctico, no una acción aislada
- quieres controlar manualmente pre-roll y post-roll

## Próximas acciones

- añade una tactical annotation
- añade un drawing
- añade el clip a una playlist
- comparte el clip cuando el contexto de revisión esté listo
