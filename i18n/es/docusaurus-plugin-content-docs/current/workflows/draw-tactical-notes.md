# Dibuja notas tácticas

Usa esto cuando el video necesita explicación visual.

Los drawings tácticos ayudan a explicar forma, movimiento, presión, distancias y decisiones alrededor de un momento del partido. Úsalos cuando un entrenador o jugador necesita ver la idea, no solo leerla.

## Usos comunes

- añadir flechas para mostrar desmarques o líneas de pase
- dibujar líneas para explicar distancias
- marcar zonas o formas de referencia
- añadir etiquetas de texto durante la preparación de revisión
- adjuntar drawings a clips para presentación

## Keyframes

Threader soporta drawing keyframes, así que una annotation puede vincularse a un momento específico del video. Usa keyframes cuando el drawing deba aparecer en un tiempo preciso y no durante un segmento amplio.

## Pasos

1. Pausa en el frame relevante.
2. Abre las herramientas de drawing.
3. Añade los elementos tácticos necesarios.
4. Vincula el drawing a un clip si pertenece a un paquete de revisión.
5. Recorre el video para confirmar que aparece cuando debe.

## Buenas prácticas

- Mantén una idea clara por drawing.
- Usa etiquetas solo cuando reduzcan ambigüedad.
- Combina drawings con clips al preparar revisión técnica.
- Usa tactical annotations cuando el mismo momento también necesita significado futbolístico estructurado.
