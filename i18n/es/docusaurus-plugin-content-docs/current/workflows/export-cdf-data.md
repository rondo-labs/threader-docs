# Exporta datos CDF

Usa esto cuando necesitas datos estructurados de eventos fuera de Threader.

La exportación CDF es una salida posible, no el único objetivo de Threader. Muchos usuarios construirán clips y playlists de revisión sin exportar datos.

## Antes de exportar

Comprueba que el proyecto tenga:

- video del partido
- mapeo de tiempo de partido si es necesario
- registros de eventos válidos
- equipos y jugadores requeridos
- advertencias de validación resueltas

## Exportar

1. Abre el flujo de exportación.
2. Revisa el feedback de validación.
3. Corrige campos obligatorios faltantes.
4. Exporta los datos CDF-ready.
5. Usa los archivos exportados en herramientas o pipelines de análisis.
