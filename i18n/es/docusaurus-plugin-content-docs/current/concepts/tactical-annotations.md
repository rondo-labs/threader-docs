# Tactical annotations

Las tactical annotations añaden significado futbolístico a clips, eventos o rangos de tiempo.

Son distintas de etiquetas libres. Deben hacer que el análisis sea consultable, revisable y compartible.

## Qué capturan

Empieza con la estructura mínima útil:

- phase
- concept
- outcome
- note opcional

Ejemplos:

- `in possession / line break / success`
- `out of possession / pressing trigger / fail`
- `defensive transition / recovery run / neutral`

## A qué se adjuntan

Adjunta tactical annotations a:

- clips, cuando el momento es material de revisión
- eventos, cuando la acción es el ancla
- rangos de tiempo, cuando la idea táctica abarca varias acciones

## Regla práctica

Usa tactical annotations cuando la pregunta sea "¿qué significa este momento?" y no solo "¿qué pasó?"
