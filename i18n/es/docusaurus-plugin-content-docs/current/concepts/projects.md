# Proyectos

Un proyecto es el contenedor local de análisis en Threader.

Usa un proyecto para un partido, sesión de entrenamiento o contexto de revisión específico. El proyecto mantiene conectados los objetos de análisis para que video, eventos, clips, annotations, drawings y playlists puedan referirse a los mismos momentos.

## Un proyecto puede contener

- referencias a videos del partido
- eventos importados o manuales
- clips
- drawings
- tactical annotations
- playlists y estructura de revisión
- estado local del proyecto

## Los datos del proyecto son locales

Los datos del proyecto son la base del flujo de trabajo de escritorio de Threader.

Los datos de colaboración en la nube, como membresía de workspace y comentarios de feed, están separados hasta que compartes o publicas material de análisis explícitamente.

## Regla práctica

Empieza con un proyecto y un video. Añade eventos, clips, annotations y playlists cuando el análisis necesite más estructura.
