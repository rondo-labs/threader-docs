# Eventos y CDF

Los eventos describen acciones o momentos futbolísticos. Pueden venir de importaciones de provider, anotación manual o futuros flujos automatizados.

CDF, Common Data Format for Football, es un formato estructurado para intercambiar datos de eventos de fútbol.

## Eventos en Threader

Los eventos pueden ayudarte a:

- navegar el timeline
- crear clips desde acciones conocidas
- filtrar momentos del partido
- validar datos de análisis
- preparar exportaciones

## CDF en Threader

CDF es importante para interoperabilidad, pero no es la única razón para usar Threader.

Puedes usar Threader para construir clips, tactical annotations, playlists y conversaciones en feed aunque no planees exportar datos CDF.

## Regla práctica

Importa datos de eventos cuando los tengas. Si no, empieza desde el video y crea clips desde rangos de tiempo.
