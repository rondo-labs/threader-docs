# 欢迎使用 Threader

**Threader** 是面向教练和分析师的桌面足球分析工作台。它把比赛视频、事件、clips、tactical annotations、playlists 和协作材料连接在同一个本地项目中。

## Threader 是什么？

Threader 帮助你从比赛画面进入结构化复盘材料：导入视频，把事件或时间范围连接到 clips，添加战术意义，构建 playlists，并在准备好时和团队分享结果。

核心流程是：

```text
Project -> Video -> Event / Time Range -> Clip -> Tactical Annotation -> Playlist / Review -> Feed
```

## 适合谁使用？

- **Performance analysts**：需要结构化比赛数据
- **Coaches**：希望把关键时刻转化为教练材料
- **Video analysts**：需要快速创建 clips 和复盘流程
- **Data analysts**：需要 CDF-compliant 事件数据进入下游 pipelines
