# Clips

Clip 是被保存下来用于分析和复盘的视频范围。

Clips 是从“观看视频”到“产出教练材料”的桥梁。它们可以从时间范围或事件创建，然后被标注、绘制、分组和分享。

## Clip 通常包含

- 开始和结束时间
- 标题
- 可用时关联事件
- 必要时关联 tactical annotations
- 有用时添加 drawings 或 notes

## 为什么 clips 重要

Clips 会让比赛时刻变成稳定对象。一个时刻成为 clip 后，你可以：

- 快速回到它
- 放进 playlists
- 添加战术意义
- 通过 feed 分享
- 用于复盘会议

## 实用规则

先尽早创建 clips。只有当 clip 值得复盘或分享时，再补充更多结构。
