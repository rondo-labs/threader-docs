# Tactical annotations

Tactical annotations 会给 clips、事件或时间范围添加足球意义。

它们不同于随意标签。它们应该让分析可以被查询、复盘和分享。

## 它们捕捉什么

从最小有用结构开始：

- phase
- concept
- outcome
- 可选 note

示例：

- `in possession / line break / success`
- `out of possession / pressing trigger / fail`
- `defensive transition / recovery run / neutral`

## 它们关联到什么

Tactical annotations 可以关联到：

- clips：当这个时刻是复盘材料
- events：当某个动作是锚点
- time ranges：当战术想法跨越多个动作

## 实用规则

当问题是“这个时刻意味着什么？”而不只是“发生了什么？”时，使用 tactical annotations。
