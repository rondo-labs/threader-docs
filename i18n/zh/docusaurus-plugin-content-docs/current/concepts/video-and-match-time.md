# 视频时间和比赛时间

视频时间和比赛时间相关，但不是同一回事。

视频时间是媒体文件中的播放位置。比赛时间是足球比赛中的时钟或真实世界时间戳，用于对齐事件和导出。

## 视频时间

视频时间用于：

- 播放
- timeline 导航
- 选择 clip 范围
- drawing keyframes

## 比赛时间

比赛时间用于：

- 导入事件需要和视频对齐时
- 事件记录需要准确时间时
- CDF 导出需要校验时间戳时
- 上半场、下半场、补时或阶段边界很重要时

## 同步

比赛时间同步会把视频位置映射到足球时间。你可以在同步完全准确前创建 clips，但结构化导出和事件校验需要可靠时间。
