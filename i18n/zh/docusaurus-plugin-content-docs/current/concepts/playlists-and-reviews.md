# Playlists 和复盘

Playlists 会把 clips 组织成复盘材料。

单个 clip 捕捉一个时刻。Playlist 会把多个时刻串成一个故事。

## 常见 playlist 类型

- 球员复盘
- 小组或位置组复盘
- 对手模式
- 比赛阶段
- 战术概念
- 训练反馈

## 好的复盘结构

好的 playlist 通常有：

- 清晰标题
- 有意安排的 clip 顺序
- 足够让别人理解主题的上下文
- 当筛选或比较很重要时，带有 tactical annotations

## 实用规则

当多个 clips 开始指向同一个教练问题时，构建 playlist。
