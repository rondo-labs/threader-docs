# Workspace 和 feed

Workspace 和 feed 是 Threader 的协作层。

当分析材料应该进入团队讨论时，使用它们。

## Workspace

Workspace 表示可以围绕共享分析流程进行协作的一组人。

Workspace 数据包括：

- 成员
- 邀请
- 账号和 profile 上下文
- 可用时的权限

## Feed

Feed 是共享分析材料变成讨论的地方。

Feed items 可以引用 clips、playlists、notes 或未来的 dashboard material 等分析产出。

## 实用规则

先做本地分析。当有值得讨论的 clip、playlist 或复盘内容时，再发布到 feed。
