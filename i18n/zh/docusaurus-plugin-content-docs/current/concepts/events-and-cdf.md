# 事件和 CDF

事件描述足球动作或比赛时刻。它们可以来自 provider 导入、手动标注，或未来的自动化流程。

CDF，即 Common Data Format for Football，是一种用于交换足球事件数据的结构化格式。

## Threader 中的事件

事件可以帮助你：

- 在 timeline 中导航
- 从已知动作创建 clips
- 筛选比赛时刻
- 校验分析数据
- 准备导出

## Threader 中的 CDF

CDF 对互操作很重要，但它不是使用 Threader 的唯一理由。

即使你不计划导出 CDF 数据，也可以用 Threader 构建 clips、tactical annotations、playlists 和 feed 讨论。

## 实用规则

有事件数据时就导入。没有时，从视频开始，用时间范围创建 clips。
