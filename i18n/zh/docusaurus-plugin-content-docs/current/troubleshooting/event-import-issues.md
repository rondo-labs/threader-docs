# 事件导入问题

当事件数据无法正常导入，或无法和比赛对齐时，查看这一页。

## 检查

- provider 或文件格式是否受支持
- 必填字段是否存在
- 球队和球员标识是否可以映射
- 事件时间是否使用预期的时钟或时间戳基础
- 导入的比赛是否和项目视频一致

## 如果事件还没准备好

你仍然可以从视频时间范围创建 clips，之后再补充事件上下文。
