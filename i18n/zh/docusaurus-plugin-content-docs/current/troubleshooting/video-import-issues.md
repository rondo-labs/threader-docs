# 视频导入问题

当视频无法导入、加载或正常播放时，查看这一页。

## 先尝试

1. 确认文件存在于本地或当前可用的硬盘上。
2. 尝试使用标准 MP4 版本。
3. 尽量避免少见编码。
4. 重新打开项目，并在提示时重新连接缺失媒体。

## 推荐格式

如果文件不稳定，把它转码成 H.264 MP4 后再导入。
