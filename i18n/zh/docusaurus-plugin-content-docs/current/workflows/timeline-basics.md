# Timeline 基础

当你想理解视频、事件、clips 和 annotations 在哪里汇合时，请阅读这一页。

Timeline 是 Threader 的中心分析表面。它不只是播放进度条，而是帮助你在比赛视频、导入事件、保存的 clips、drawings 和战术意义之间移动。

## Timeline 连接什么

- 视频播放位置
- 比赛事件
- 选中的时间范围
- clips
- tactical annotations
- 复盘时刻

## 常见动作

- 跳转到视频中的某一刻
- 查看当前 playhead 附近的事件
- 选择时间范围
- 从选择范围创建 clip
- 从事件创建 clip
- 在构建 playlist 前复查密集片段

## 推荐的第一个流程

1. 导入比赛视频。
2. 移动到一个重要时刻。
3. 选择有用的时间范围。
4. 创建 clip。
5. 如果这个 clip 会被复盘或分享，添加战术意义。

导入事件会让 timeline 更丰富，但创建第一个 clip 并不依赖事件数据。
