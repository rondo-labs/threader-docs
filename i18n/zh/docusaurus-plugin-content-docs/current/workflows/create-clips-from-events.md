# 从事件创建 clips

当导入或手动创建的事件已经标记了你要复盘的时刻时，使用这个方式。

基于事件的 clip 会保持和结构化比赛数据的连接。

## 步骤

1. 从 timeline 或事件列表中选择一个事件。
2. 从该事件创建 clip。
3. 围绕事件调整 clip 范围。
4. 确认标题和已关联事件。
5. 如果这个时刻需要足球意义，添加 tactical annotation。

## 适合的场景

- provider 数据已经标记了该动作
- 事件本身就是复盘锚点
- 之后需要按事件类型筛选或分组 clips
- 你计划导出或校验结构化数据

## 范围建议

不要把每个事件 clip 都裁得太紧。教练复盘通常需要动作发生前的铺垫，以及动作之后的结果。
