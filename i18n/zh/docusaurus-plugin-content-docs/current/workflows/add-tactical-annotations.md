# 添加 tactical annotations

当一个 clip 或时刻需要的不只是事件标签，而是足球意义时，添加 tactical annotation。

Tactical annotations 会把 clips、事件或时间范围连接到结构化足球概念。这样这些时刻就更容易筛选、复盘和讨论。

## 最小有用 annotation

先从小结构开始：

- phase
- concept
- outcome
- 必要时添加简短 note

在 annotation 变得有用之前，不需要填满所有细节。

## 应该关联到什么

把 annotation 关联到最有分析价值的对象：

- 关联到 clip：当这个时刻是复盘材料
- 关联到 event：当某个具体动作是锚点
- 关联到 time range：当战术想法跨越多个动作

## 好的 annotation notes

好的 note 会解释为什么这个时刻重要：

- `pressing trigger after backward pass`
- `fullback receives isolated with no support`
- `counter-press fails after loose second ball`

## 添加之后

你可以用 annotations 来：

- 按战术概念筛选 clips
- 围绕重复模式构建 playlists
- 发布带有更多上下文的复盘材料
- 比较成功和失败的结果
