# 导出 CDF 数据

当你需要把结构化足球事件数据带到 Threader 外部时，使用 CDF 导出。

CDF 导出是一条输出路径，但不是 Threader 的唯一目标。很多用户会只构建 clips 和复盘 playlists，而不导出数据。

## 导出前

检查项目是否包含：

- 比赛视频
- 必要时已设置比赛时间映射
- 有效事件记录
- 必需的球队和球员信息
- 已解决的校验警告

## 导出

1. 打开导出流程。
2. 查看校验反馈。
3. 修复缺失的必填字段。
4. 导出 CDF-ready 数据。
5. 在下游分析工具或 pipeline 中使用导出文件。
