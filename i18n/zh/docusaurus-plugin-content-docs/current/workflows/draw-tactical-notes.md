# 绘制战术说明

当视频需要视觉解释时，使用 drawings 或 telestration。

战术 drawing 可以解释阵型、移动、压迫、间距和某个比赛时刻中的决策。当教练或球员需要“看见”这个想法，而不只是阅读文字时，它很有用。

## 常见用途

- 用箭头展示跑动或传球线路
- 用线条解释站位距离
- 标记区域或参考形状
- 在复盘准备中添加文字标签
- 把 drawing 关联到 clips 以便展示

## Keyframes

Threader 支持 drawing keyframes，因此一个 annotation 可以绑定到视频中的具体时刻。当 drawing 只应出现在精确时间点，而不是整个片段中时，使用 keyframes。

## 步骤

1. 暂停在相关画面。
2. 打开 drawing 工具。
3. 添加需要的战术元素。
4. 如果 drawing 属于复盘材料，把它关联到 clip。
5. 拖动视频检查 drawing 是否在正确时间出现。

## 最佳实践

- 每个 drawing 保持一个清晰想法。
- 只有在能减少歧义时才使用文字标签。
- 准备教练复盘时，把 drawings 和 clips 配对。
- 当同一个时刻还需要结构化足球意义时，使用 tactical annotations。
