# 从时间范围创建 clips

当你知道视频中的关键时刻，但没有具体事件作为起点时，使用这个方式。

时间范围 clip 是从看视频到保存复盘材料的最快路径。

## 步骤

1. 在视频中找到关键时刻。
2. 在 timeline 上选择开始和结束范围。
3. 从选择范围创建 clip。
4. 调整开始和结束时间。
5. 添加清晰标题。

## 适合的场景

- 你没有导入事件数据
- 这个时刻跨越多个事件
- 你复盘的是战术模式，而不是单个动作
- 你想手动控制 pre-roll 和 post-roll
