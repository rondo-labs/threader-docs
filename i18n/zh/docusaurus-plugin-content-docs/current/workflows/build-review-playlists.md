# 构建复盘 playlists

当多个 clips 属于同一个教练或分析故事时，创建 playlist。

Playlist 会把单独的 clips 变成复盘材料。它可以按球员、阶段、战术概念、对手行为或会议主题组织。

## 步骤

1. 选择属于同一主题的 clips。
2. 创建 playlist。
3. 给 playlist 一个清晰复盘标题。
4. 按你想展示的顺序重新排列 clips。
5. 在需要的位置添加 notes 或上下文。

## 好的 playlist 标题

- `High press recoveries`
- `Final-third cutbacks`
- `Player 8 receiving under pressure`
- `Defensive transition problems`

## 分享前检查

确认每个 clip 都有：

- 清晰范围
- 可读标题
- 必要时有战术意义
- 足够让别人理解的上下文
