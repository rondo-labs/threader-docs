# Annotation 快捷键

当你需要反复添加结构化足球意义时，使用 annotation 快捷键。

## 常见 annotation 任务

- 打开 tactical annotation editor
- 确认当前 annotation
- 在字段之间移动
- 可用时复用最近值
- 回到 clip 或 timeline 上下文

## 提示

不要给每个时刻过度标注。只有当结构化战术意义能帮助筛选、复盘或分享时，再添加它。
