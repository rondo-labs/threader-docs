# 创建或打开项目

当你要开始一次比赛分析时，请先创建或打开项目。

Threader 项目是一套本地工作空间。它把视频、导入或手动创建的事件、clips、drawings、tactical annotations、playlists 和项目状态放在同一个分析上下文中。

## 创建新项目

1. 启动 Threader。
2. 选择新建项目。
3. 用比赛、训练课或复盘主题命名项目。
4. 把项目保存在适合存放分析文件的位置。

当分析依赖比赛视频、比赛时间或事件数据时，通常一场比赛使用一个项目。

## 打开已有项目

回到正在进行的分析时，优先使用最近项目。

如果项目能打开但媒体缺失，项目数据通常仍然存在；Threader 可能只是需要你重新连接移动过的视频文件。

## 项目中包含什么

- 比赛视频引用
- 事件数据
- clips
- drawings
- tactical annotations
- playlists 和复盘结构
- 本地分析状态

## 什么不只属于项目

Workspace 成员、feed 评论、邀请和账号资料属于协作层。

项目是本地分析容器。Workspace 和 feed 是明确的分享表面。
