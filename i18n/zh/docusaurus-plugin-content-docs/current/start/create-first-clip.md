# 创建第一个 clip

当你发现一个应该进入复盘的时刻时，创建 clip。

Clip 是一个被保存的视频范围，并带有足够上下文，方便之后回看、标注、组织和分享。

## 从时间范围创建

1. 在视频中找到关键时刻。
2. 在 timeline 上选择开始和结束范围。
3. 从选择范围创建 clip。
4. 如有需要，调整范围。
5. 给 clip 一个清晰标题。

即使项目没有导入事件数据，也可以使用这个方式。

## 从事件创建

1. 在 timeline 或事件列表中选择一个事件。
2. 从该事件创建 clip。
3. 根据需要调整事件前后的 pre-roll 和 post-roll。
4. 添加能解释重点的标题。

## 创建之后

下一步可以：

- 添加 tactical annotation
- 添加 drawing 或 telestration
- 加入 playlist
- 准备好复盘上下文后，发布 clip 或 playlist 到 feed

好的 clip 标题应该描述足球想法，而不只是事件类型。
