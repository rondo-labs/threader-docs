# 导入事件数据

当你已经有来自 provider、数据文件或其他流程的比赛事件时，使用这个流程。

导入事件可以把 timeline 上的比赛时刻和视频连接起来。事件可以作为 clip 起点、筛选条件和复盘上下文。

## 导入事件

1. 打开包含比赛视频的项目。
2. 选择事件导入动作。
3. 选择支持的 provider 或文件格式。
4. 检查映射或校验反馈。
5. 确认导入。

## 导入之后

你可以用导入事件来：

- 跳转到 timeline 上的关键时刻
- 从选中事件创建 clips
- 筛选密集片段
- 把 tactical annotations 连接到具体比赛动作
- 在需要时准备 CDF 导出

## 如果你没有事件数据

仍然可以分析视频。

先从时间范围创建 clips，之后再补充 tactical annotations 或手动事件上下文。
