# 导入比赛视频

当项目还没有视频时，请从这里开始。

视频为 Threader 提供分析表面。事件、clips、drawings 和 annotations 只有和比赛画面保持连接时才最容易理解。

## 导入视频

1. 打开或创建项目。
2. 选择视频导入动作。
3. 选择本地比赛视频文件。
4. 等待 Threader 读取元数据。
5. 确认视频可以正常播放。

Threader 面向常见比赛视频格式设计，例如 MP4、MOV 和 MKV。

## 如果导入失败

把视频转码成标准 H.264 MP4 后重试。

超大文件、少见编码，或存放在不可用外接硬盘上的文件，都可能导致播放或元数据问题。

## 导入之后

下一步可以：

- 用播放器和 timeline 定位视频
- 如果已有事件数据，导入事件
- 选择时间范围并创建 clip
- 当需要 CDF 导出时，再设置比赛时间同步

创建第一个 clip 不需要先有事件数据。
